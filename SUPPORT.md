# Support

## Asking a question

For general questions and usage help, open a [GitHub Discussion](https://github.com/celticblast/string-utils/discussions).

## Reporting a bug

Open a [GitHub Issue](https://github.com/celticblast/string-utils/issues/new?template=bug_report.md) using the bug report template.

## Requesting a feature

Open a [GitHub Issue](https://github.com/celticblast/string-utils/issues/new?template=feature_request.md) using the feature request template.

## Reporting a security vulnerability

See [SECURITY.md](./SECURITY.md) for how to report security issues privately.
